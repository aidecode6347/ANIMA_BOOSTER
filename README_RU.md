# ANIMA_BOOSTER (BSS)

**ANIMA_BOOSTER (BSS)** — это высокотехнологичный пакет оптимизации для модели **Anima DiT 2B** в ComfyUI. Разработан для обеспечения максимальной производительности, снижения потребления видеопамяти (VRAM) и ускорения генерации на видеокартах NVIDIA.

Этот пакет позволяет получить **суммарное ускорение от 3.5× до 5.0×** по сравнению со стандартным воркфлоу Anima в ComfyUI без потери визуального качества.

## ⚡ Ключевые возможности

* **Интегрированная JIT-компиляция (`torch.compile`):** Безопасная компиляция блоков DiT в один клик непосредственно в загрузчике. Работает на стабильном бэкенде `inductor` без использования CUDA Graphs, обеспечивая стабильность и прирост скорости на **20–40%**.
* **SageAttention:** Встроенная поддержка сверхбыстрого 8-битного внимания для моделей DiT, значительно ускоряющая вычисления при сниженном потреблении памяти.
* **Адаптивный TeaCache:** Интеллектуальное кэширование скрытых состояний. Динамически регулирует шаг кэширования: снижает его на ранних шагах для сохранения геометрии и повышает на поздних шагах для максимального ускорения.
* **BSS Premium UI:** Интегрированная темная высококонтрастная тема оформления для нод пакета. Включает полностью переработанное управление слайдерами по всей ширине, адаптивное скрытие полей для неактивных пресетов и полную блокировку назойливых всплывающих подсказок (tooltips) для комфортной работы.

## 📥 Установка

### Способ 1: Через ComfyUI Manager (Рекомендуется)
1. Откройте ComfyUI и нажмите кнопку **Manager**.
2. Нажмите **Install via Git URL**.
3. Вставьте ссылку на этот репозиторий: `https://github.com/BlackSnowSkill/ANIMA_BOOSTER`

1. Откройте терминал и перейдите в папку кастомных нод ComfyUI:
   ```bash

   ```

   ```bash
   git clone https://github.com/BlackSnowSkill/ANIMA_BOOSTER.git
   ```
3. Перезапустите ComfyUI.

## 🧩 Список нод пакета `BSS/AnimaBooster`

Все ноды зарегистрированы в категории `BSS/AnimaBooster`:

1. 📥 **Anima Booster Loader (BSS)** (класс `AnimaBoosterLoader`)
   - Загружает модель Anima DiT в оптимизированном формате fp16.
   - **SageAttention**: Автоматически применяет ускоренное 8-битное внимание при наличии в системе. При отсутствии — бесшовно переключается на встроенный PyTorch SDPA.
   - **Torch Compile**: Встроенный тумблер для безопасной JIT-компиляции блоков трансформера.
2. 🎛️ **Anima TeaCache (BSS)** (класс `AnimaTeaCache`)
   - Реализует адаптивное кэширование скрытых состояний на основе шагов очистки шума (TeaCache).
   - Автоматически рассчитывает шкалу таймстепов для любых типов семплеров (включая SDE).
3. 🖼️ **Anima Latent Image (BSS)** (класс `AnimaLatentImage`)
   - Утилита для генерации пустых латентов с автоматическим выравниванием размеров под патчи Anima DiT (2x2), предотвращающая ошибки несоответствия размерностей тензоров. Предоставляет готовые пресеты соотношений сторон.

## 📈 Адаптивный порог TeaCache (Adaptive TeaCache)

В отличие от стандартных реализаций TeaCache с фиксированным порогом (threshold), версия от **BSS** использует **динамический адаптивный порог**, меняющийся в процессе денойзинга:
- **На ранних шагах** (высокий уровень шума, формируется структура изображения) порог снижается автоматически для обеспечения максимальной точности прорисовки геометрии.
- **На поздних шагах** (детали стабилизировались, идет микро-текстурирование) порог повышается, позволяя пропускать до 80% вычислений блоков без потери качества.

## 🛠️ Установка и зависимости на Windows

Все ресурсоемкие библиотеки оптимизации (**SageAttention** и JIT-компилятор **Triton**) являются **полностью опциональными**. Пакет спроектирован по принципу *Graceful Degradation*: если библиотеки или компиляторы не установлены, ноды автоматически отключают оптимизации, бесшовно переходя на стандартные механизмы PyTorch, что гарантирует работу без сбоев и "крашей" ComfyUI.

> [!IMPORTANT]
> **Triton необходим как для работы SageAttention, так и для JIT-компиляции (`torch.compile`)**!
> Если вы хотите включить `torch_compile` (дающую до +40% к скорости генерации) или использовать SageAttention на Windows, вам **необходимо** установить `triton-windows`. Без него JIT-компиляция будет автоматически и безопасно отключена с выводом предупреждения в консоль, чтобы избежать падений.

### 📦 Установка Triton и SageAttention (для Windows & Portable Build)

Портативные сборки ComfyUI (использующие изолированный `python_embeded` на Python 3.13 или 3.12) не имеют установленных инструментов компиляции C++ (MSVC / Build Tools). Из-за этого стандартная команда `pip install sageattention` выдает ошибку компиляции.

Вы можете легко обойти эту проблему одним из следующих способов:

#### Способ 1: Автоматическая установка через ComfyUI Manager (Рекомендуется)
1. В ComfyUI Manager найдите и установите ноду **ComfyUI-Sage-EasyInstall**.
2. Перезагрузите ComfyUI.
3. Данная нода автоматически проанализирует вашу систему, установит нужную версию `triton-windows` и загрузит скомпилированный бинарный пакет (`.whl` wheel) SageAttention, разработанный специально под вашу версию Python и CUDA.

#### Способ 2: Ручная установка предкомпилированных Wheel-файлов
Если вы предпочитаете установить всё вручную в портативную сборку:
1. Откройте командную строку (CMD или PowerShell) в вашей главной папке ComfyUI.
2. Установите Triton для Windows:
   ```bash
   .\python_embeded\python.exe -m pip install triton-windows
   ```
3. Скачайте предкомпилированный файл `.whl` для SageAttention, соответствующий вашей версии Python (например, `cp313` для Python 3.13) и CUDA (например, `cu124` / `cu128`):
   - Готовые сборки можно найти в релизах репозитория: **[sdbds/SageAttention-for-windows](https://github.com/sdbds/SageAttention-for-windows)**.
   - Также предкомпилированные колеса публикуются в проекте: **[wildminder/AI-windows-whl](https://github.com/wildminder/AI-windows-whl)**.
4. Установите скачанный файл в портативное окружение:
   ```bash
   .\python_embeded\python.exe -m pip install <путь_к_скачанному_файлу.whl>
   ```

## 🧑‍💻 Технические детали реализации для разработчиков

- **Основа модели**: Anima DiT базируется на архитектуре `MiniTrainDIT` с адаптером `LLMAdapter`.
- **Точка интеграции SageAttention**: Патч накладывается через стандартный словарь параметров `transformer_options["optimized_attention_override"]`.
- **Точка интеграции Torch Compile**: Вызов встроенной функции `set_torch_compile_wrapper` из `comfy_api.torch_helpers` на уровне индивидуальных блоков трансформера (что обеспечивает LoRA-совместимость и снижает оверхед компиляции).
- **Изоляция**: Все изменения структуры графа и весов производятся исключительно на клонированной копии модели (`model.clone()`). Это исключает конфликты и оставляет оригинальную модель в кэше ComfyUI нетронутой.
